# Tecnologia, IA Generativa e Lovable

O projeto foi desenvolvido integralmente utilizando a plataforma **Lovable**,
com apoio de **IA Generativa baseada em prompts**.

## Abordagem adotada
- Criação da interface e componentes via IA
- Definição da arquitetura do projeto por prompts
- Geração e refinamento de conteúdo educacional
- Iterações orientadas por feedback visual e funcional

## Stack gerada pela plataforma
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Radix UI
- Lucide React
- LocalStorage API

O foco do projeto esteve na **experiência educacional e no design pedagógico**,
e não na escrita manual de código do zero.
