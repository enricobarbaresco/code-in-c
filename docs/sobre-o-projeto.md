# Sobre o Projeto

O Code in C nasceu com a proposta de facilitar o primeiro contato de estudantes
com a programação, especialmente com a linguagem C, que costuma ser um desafio
para quem está começando.

A plataforma foi desenvolvida utilizando a ferramenta Lovable, permitindo focar
na experiência educacional, organização do conteúdo e progressão do aprendizado.

O projeto prioriza clareza, exemplos simples e evolução gradual de dificuldade.
