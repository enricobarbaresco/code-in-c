# Metodologia de Educacional

A metodologia do Code in C é baseada em três pilares:

## 1. Progressão gradual
O conteúdo é organizado do básico ao avançado, evitando sobrecarregar o estudante
logo no início.

## 2. Aprendizado conceitual + prático
Cada conceito apresentado é acompanhado de exemplos e propostas de aplicação.

## 3. Estímulo à prática
O projeto incentiva o aprendizado ativo por meio de desafios e pequenos projetos,
fundamentais para a fixação do conteúdo.
