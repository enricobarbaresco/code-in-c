# Sistema de Gamificação

O Code in C utiliza gamificação para aumentar o engajamento do usuário.

## Sistema de XP
- Completar tópico: +50 XP
- Quiz: pontuação × 10
- Progressão de nível a cada 500 XP

## Conquistas
O sistema conta com 13 conquistas, incluindo:
- Primeiro programa
- Mestre dos loops
- Mago dos ponteiros
- Detetive de bugs
- Programador avançado
