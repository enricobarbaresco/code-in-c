Um **algoritmo** é uma sequência de passos ordenados para resolver um problema. Você já usa algoritmos no dia a dia sem perceber!

Por exemplo, o algoritmo para fazer um café:
1. Pegue a cafeteira
2. Coloque água
3. Adicione o pó de café
4. Ligue a cafeteira
5. Espere o café ficar pronto
6. Sirva na xícara

Na programação, escrevemos algoritmos usando uma linguagem que o computador entende. A diferença é que precisamos ser **muito mais específicos** - o computador não sabe o que é "esperar ficar pronto" a menos que digamos exatamente quanto tempo.

**Características de um bom algoritmo:**
- **Finitude**: Deve ter um fim claro
- **Definição**: Cada passo deve ser claro e sem ambiguidades
- **Entrada**: Pode receber dados externos
- **Saída**: Deve produzir algum resultado
- **Efetividade**: Cada passo deve ser simples o suficiente para ser executado
