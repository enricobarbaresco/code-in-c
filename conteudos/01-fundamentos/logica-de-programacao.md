Programação é a arte de escrever instruções que um computador pode entender e executar. Pense no computador como um assistente muito obediente, mas que precisa de instruções extremamente precisas.

Quando você programa, você está criando um conjunto de passos (chamado de **algoritmo**) que o computador seguirá exatamente como você escreveu. Se você pedir para ele somar 2 + 2, ele fará isso. Se você escrever errado, ele fará errado também!

A linguagem C é uma das linguagens de programação mais antigas e importantes. Criada em 1972 por Dennis Ritchie nos laboratórios Bell, ela é usada até hoje para:
- Sistemas operacionais (Windows, Linux, macOS)
- Jogos e motores gráficos
- Sistemas embarcados (carros, eletrodomésticos, aviões)
- Compiladores de outras linguagens
- Bancos de dados e servidores

A lógica de programação é a forma de organizar o pensamento para resolver
problemas de maneira estruturada.

Ela envolve:
- Entender o problema
- Definir entradas
- Processar informações
- Gerar uma saída

Aprender lógica é essencial antes mesmo de focar em uma linguagem específica.
