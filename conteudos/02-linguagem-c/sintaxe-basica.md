Todo programa em C tem uma estrutura básica que você precisa conhecer. O ponto de entrada é sempre a função **main()** - é por onde o computador começa a executar seu código.

A estrutura básica inclui:
- **Diretivas de pré-processador**: Começam com # e são processadas antes da compilação
- **Declarações globais**: Variáveis e funções disponíveis em todo o programa
- **Função main()**: Ponto de entrada obrigatório
- **Corpo das funções**: Código que será executado

# Sintaxe Básica da Linguagem C

A linguagem C possui uma sintaxe estruturada e direta.

Características importantes:
- Uso de ponto e vírgula no final das instruções
- Blocos de código delimitados por chaves { }
- Função principal chamada main()

Entender a sintaxe é fundamental para evitar erros comuns no início.

EXEMPLO DE CÓDIGO:

#include <stdio.h>

int main() {
    // Seu código vai aqui
    printf("Olá, Mundo!\n");
    return 0;
}
